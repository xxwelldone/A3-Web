About us

Good Enough was designed and built by student of Brazil's Anhembi Morumbi University for academic reasons. Every semester students of this university has to implement a project based on what was learnt in the respective semester.

This semester we were tasked with two projects, one being a Java back-end project and the other being a web and mobile application with HTML, CSS and JavaScript consuming an API and database methods.
