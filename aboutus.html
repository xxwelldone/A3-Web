About us

"Good Enough" was created by students from Anhembi Morumbi University in Brazil as part of their academic curriculum. Each semester, students are assigned projects that reflect the knowledge they have gained during that period.

In this semester, we were given two projects. One project focused on developing a Java back-end system, while the other involved creating a web and mobile application using HTML, CSS, and JavaScript. The application interacted with an API and utilized database methods to enhance its functionality and effectiveness.
